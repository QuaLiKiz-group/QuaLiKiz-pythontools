§ profile_job()

def qualikiz_tools.machine_specific.craypat.profile_job (   job)

Read QuaLiKiz Profiling info This function polls for the QuaLiKiz profiling information generated with CrayPat.

It needs the metadata containing the jobdata. This is always generated if QuaLiKiz was run with one of the pythontools scripts. For example, with 'pythontools.py inputgo runs/mini'

    - path: Path to poll. Path should contain a metadata
            and job file.
Returns
  • A pair containing (jobnumber, tables), with:
    • jobnumber: the jobnumber as read from the metadata
    • All tables read from craypat. Right now the first table contains the sampling of line-numbers and the second table contains the sampling of functions
